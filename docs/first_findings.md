# First Findings

最終更新: 2026-07-02

このメモは、OpenAI embedding による最初の意味空間解析の観察記録です。結論ではなく、次に精読・照合すべき箇所を探すための作業ノートとして扱います。

## 実行条件

- 対象: `data/hyakunin_isshu.csv` の小倉百人一首 100首。
- 本文データ: Japanese Wikisource「小倉百人一首」固定版 oldid 240235 に基づく初期候補。
- 入力形式: `original_kana`。正規化後の漢字かな交じり本文と読みを改行で結合。
- モデル: OpenAI `text-embedding-3-large`。
- 次元数: 3072。
- PCA: centered Gram matrix + power iteration による純 Python 実装。
- 隣接比較: 小倉順の99隣接ペアと、ランダム順10,000回。
- ランダム seed: `20260702`。
- 生成出力: `public/data/embeddings_pca.json`, `public/data/adjacency_stats.json`, `public/data/pair_rankings.json`, `docs/viewer/`, `docs/figures/semantic-pca-map.svg`, `docs/figures/semantic-pca-distribution.svg`。このうち `public/data/*.json` と `docs/viewer/` は、現行の日本語HTML読み物公開ではGitHub公開対象に含めない。
- 非公開出力: `data/embeddings/` 以下の embedding vector cache。

## 観察1: 第1主成分は恋・身の嘆きと季節・自然景をゆるく分ける

PCA の寄与率は PC1 が 0.0624、PC2 が 0.0525 であり、2次元図だけで構造を強く断定するには小さい。ただし、PC1 の方向には主題の偏りが見える。

主題別 centroid:

| 主題 | 首数 | PC1平均 | PC2平均 |
|---|---:|---:|---:|
| 恋 | 43 | 0.1220 | -0.0138 |
| 雑 | 19 | 0.0349 | 0.0072 |
| 秋 | 17 | -0.1641 | 0.0063 |
| 冬 | 6 | -0.1980 | 0.0432 |
| 春 | 6 | -0.1018 | 0.0372 |
| 夏 | 4 | -0.2321 | -0.1614 |
| 羇旅 | 4 | -0.0599 | 0.1166 |
| 離別 | 1 | -0.1518 | 0.0469 |

PC1 が低い側には、7 阿倍仲麻呂、31 坂上是則、2 持統天皇、87 寂蓮法師、79 左京大夫顕輔、71 大納言経信など、季節・自然景・旅に寄る歌が多い。

PC1 が高い側には、99 後鳥羽院、38 右近、45 謙徳公、44 中納言朝忠、50 藤原義孝、20 元良親王など、恋、嘆き、身の内面に寄る歌が多い。

要検証: これは「歌の主題そのもの」を直接抽出したというより、短歌本文と読みからモデルが捉えた語彙・表現上の近さである。掛詞、詞書、出典文脈、歌枕はまだ十分に入っていない。

## 観察2: 小倉順の平均隣接類似度はランダム順とほぼ同じ

小倉順99ペアの平均 cosine similarity は 0.4053、中央値は 0.4056 だった。ランダム順10,000回の平均は 0.4047、標準偏差は 0.0061 である。

| 指標 | 値 |
|---|---:|
| 小倉順 平均 | 0.4053 |
| 小倉順 中央値 | 0.4056 |
| ランダム順 平均 | 0.4047 |
| ランダム順 SD | 0.0061 |
| z-score | 0.0968 |
| percentile | 0.5447 |

この条件では、「小倉順は全体としてランダムより明確に近い」とは言いにくい。むしろ、全体平均よりも、局所的に強い連続と大きな転換がどこにあるかを見る方がよい。

## 観察5: 世間的に論点になりやすい箇所の見え方

成立論・百人秀歌比較で論点になりやすい後半部については、いくつか初期観察がある。

- 99 後鳥羽院は PC1 の高い側の極端な点にあり、恋・嘆き・身の内面側に強く寄る。
- 98 従二位家隆 - 99 後鳥羽院は大きなジャンプである。
- 99 後鳥羽院と 100 順徳院は、今回の embedding では特に強い意味的隣接ペアにはならない。
- 93 鎌倉右大臣 - 94 参議雅経も大きなジャンプであり、鎌倉右大臣周辺の置かれ方は後で精読したい。

これは、後鳥羽院・順徳院や鎌倉右大臣の配置に歴史的意図があると主張するものではない。むしろ、先行研究、百人秀歌、出典部立、歌人年代と重ねて確認する候補点である。

## 観察7: `text-embedding-3-small` との比較

同じ `original_kana` 入力で `text-embedding-3-small` も実行した。`large` の図より劇的に綺麗に主題分離するわけではないが、小倉順の平均隣接類似度はランダム順との比較でやや高く出た。

| 指標 | large | small |
|---|---:|---:|
| 次元数 | 3072 | 1536 |
| PC1+PC2寄与率 | 0.1149 | 0.1168 |
| 同主題ペア平均類似度 | 0.4161 | 0.4215 |
| 異主題ペア平均類似度 | 0.4010 | 0.4032 |
| 同主題 - 異主題 gap | 0.0151 | 0.0184 |
| 最近傍が同主題の率 | 0.420 | 0.400 |
| 上位5近傍の同主題率 | 0.448 | 0.426 |
| 小倉順平均隣接類似度 | 0.4053 | 0.4146 |
| ランダム順比較 z-score | 0.0968 | 1.149 |
| ランダム順比較 percentile | 0.5447 | 0.8721 |

`small` で共通して大きなジャンプとして残ったもの:

- 9 小野小町 - 10 蝉丸。
- 41 壬生忠見 - 42 清原元輔。
- 68 三条院 - 69 能因法師。
- 93 鎌倉右大臣 - 94 参議雅経。
- 98 従二位家隆 - 99 後鳥羽院。

`small` で共通して隣接類似度上位に残ったもの:

- H001 天智天皇 - H002 持統天皇。
- H006 中納言家持 - H007 阿倍仲麻呂。
- H052 藤原道信朝臣 - H053 右大将道綱母。
- H067 周防内侍 - H068 三条院。
- H070 良暹法師 - H071 大納言経信。
- H085 俊恵法師 - H086 西行法師。

現時点の読み:

- `small` の方が、小倉順の隣接平均では少し「順番らしさ」を出す。
- 一方、近傍が同じ主題になる率では `large` がわずかに高い。
- したがって「small の方が良い」とはまだ言わず、`large/small` の両方で安定する近接・ジャンプを重視する。
- 特に H093-H094 と H098-H099 は両モデルで大ジャンプとして残るため、後半部の精読候補として強い。

## 現時点の限界

- UMAP は未実行。依存パッケージを入れず、標準ライブラリだけで PCA を実装した。
- 百人秀歌順は未入力。小倉順との差分はまだ測っていない。
- 出典勅撰集の巻・歌番号・部立は初期候補であり、要照合。
- 主題タグは最小タグであり、複数タグ・細分類が不足している。
- 現代語訳・注釈的パラフレーズはまだ使っていない。
- embedding 結果はモデル依存であり、古語短文への適性を別モデルと比較する必要がある。
