## 百人秀歌データ

百人秀歌は小倉百人一首と歌群が近い一方、配列と一部歌人・歌が異なります。公開データ候補は未確定です。

初期方針:

- まずは百人秀歌順、対応する小倉番号、差し替え歌、variant_groupだけの最小CSVを作る。
- 本文は出典が確認できるまで空欄または非公開メモに留める。
- 配列差分の可視化は、本文なしでも順序データだけで初期実装できる。

## 採用判断メモ

Milestone 1で必要な最小データは、次の条件を満たす必要があります。

- 100首すべての歌番号、歌人名、本文、読み、出典、部立が揃う。
- 本文の採用元と校訂方針が明記されている。
- 出典情報の根拠が本文サイトだけでなく、信頼できる注釈書・勅撰集索引・研究資料で確認できる。
- 公開データに本文を含める場合、再配布可能性が確認されている。

未確定の間は、生成用の `public/data/*.json` に本文を含めず、歌番号・歌人名・主題タグなどの派生情報だけを出力します。ただし現行の日本語HTML読み物公開では、`public/data/*.json` 自体をGitHub公開対象に含めません。読者向けに必要な対応表や歌本文は、出典・利用条件を明記したうえでHTML本文または参照資料ページに置きます。
