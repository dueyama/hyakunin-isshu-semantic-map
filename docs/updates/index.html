更新履歴

GitHubには細かなコミット履歴が残る。このページでは、そのすべてを写すのではなく、読者に意味のある変更を概要として記録する。

まだ公開前のため、更新履歴は空欄にしておく。公開後は、細かな修正や作業単位はGitHubのコミット履歴に残し、このページには本文の追加、数値の変更、図の差し替え、解釈の修正など、読み手に知らせるべき概要をまとめる。

公開後に記録

初回公開以後の主な追加・修正をここに記録する。必要に応じて、該当するGitHubコミットやリリースへのリンクも添える。
